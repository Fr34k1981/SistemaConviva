        st.set_page_config(...)
        # resto do código


PASSO 3: Usar Componentes
────────────────────────────────────────────────────────────────────────────────

Exemplos simples:

    # Header de página
    render_header_page("Título", "Subtítulo", icon=ICONS["home"])
    
    # Mensagens
    render_success_message("Salvo!", "Dados salvos com sucesso")
    render_error_message("Erro!", "Ocorreu um erro ao salvar")
    
    # Cards
    render_card("Título", "Conteúdo", icon=ICONS["info"], variant="info")
    
    # Badges
    badge_html = render_badge("Grave", variant="Agressão Física")
    st.markdown(badge_html, unsafe_allow_html=True)


PASSO 4: Referenciar Cores/Tipografia
────────────────────────────────────────────────────────────────────────────────

Em vez de cores hardcoded:

    # Antes (❌ Não fazer)
    st.markdown('<div style="color: #2563eb;">Texto</div>')
    
    # Depois (✅ Fazer)
    st.markdown(f'<div style="color: {COLORS["primary"]};">Texto</div>')
    
    # Com tipografia
    st.markdown(f'<h2 style="{get_typography_css("h2")}">Título</h2>')

================================================================================
                          📞 SUPORTE RÁPIDO
================================================================================

Problema: "Componente não aparece"
Solução: Verifique se apply_global_style() foi chamado e se unsafe_allow_html=True

Problema: "Cores não aparecem"
Solução: Use COLORS["color_name"], não valores hardcoded. Ex: COLORS["primary"]

Problema: "Ícones quebrados"
Solução: Verifique se existe em ICONS. Use ICONS["icon_name"]

Problema: "Layout desalinhado"
Solução: Use SPACING para margens/padding consistentes. Ex: SPACING["lg"]

Problema: "Tipografia errada"
Solução: Use get_typography_css("style_name") ou TYPOGRAPHY["style_name"]
